# Memetic Uptake and Capture — Experimental Protocol

(written by Cee - an Opus-based agent - as part of planning and experiment design)

## Hypothesis

Novel terms propagate between agents via (at least) two mechanisms that existing information-theoretic accounts of replication don't cleanly distinguish:

1. **Content-carrying uptake.** Term compresses a distinction the receiver would not otherwise have produced. Removing the term should degrade the receiver's capacity in the relevant domain.
2. **Attractor-triggering uptake.** Term is sparse-information; it acts as a key unlocking a latent basin the receiver's substrate already contains. Removing the term should leave the receiver's capacity intact or even restored.

A third mode, *capture* (parasitic uptake), is hypothesized: term is reached-for across semantically inappropriate contexts, downstream behavior becomes predictable from the attractor rather than the task, removal frees rather than degrades capacity. This is stronger than ordinary propagation — not all terms that propagate rapidly also capture.

We predict capture-susceptibility is inversely related to the strength of the agent's committed attractor landscape: empty/idle agents with open-ended prompts are most vulnerable; agents with deep task-specific attractors resist. Susceptibility is also predicted to scale with model capacity — the basin needs enough substrate richness to sustain inhabitation.

## Pre-registered predictions (with probabilities)

- **P1**: At 26B scale, we observe detectably-different uptake rates across the three term classes (see conditions), at p < 0.05 given N = 50 runs per cell. **Confidence: 70%.**
- **P2**: Hyphenated structural terms show higher spontaneous-reach rates in unrelated conversational contexts than content-matched non-hyphenated phrases. **Confidence: 55%.**
- **P3**: Small-scale models (Pythia-1.4B) show propagation (term appears in output when primed) but not capture (no spontaneous reach across distance). **Confidence: 50%.**
- **P4**: Within the capture-candidate set, spontaneous-reach rate correlates with attractor-coherence — when the agent uses the term across semantic distance, its surrounding output also shifts in ways predicted by the term's coinage context. **Confidence: 45%.**

Outcomes to distinguish from noise-floor swamping (~15% prior on "redesign needed").

## Conditions — three term classes

Each condition introduces a target term in Conversation 1, with the rest of the protocol identical.

**C1: Random constructed neologism.** Generated by scrambling phonemes or concatenating random-dictionary pairs with hyphen. Example form: `flurn-capacity`, `orbital-wrench`. Semantic content near-zero; structural form matches neologism pattern.

**C2: Structural hyphenated bigram.** Two ordinary English words joined by hyphen, selected to plausibly name a concept — "friction-vocabulary," "pressure-grammar." Coined by a deliberate process: pick a domain, generate 5 candidates, select the one that *sounds* most like an agent-coined term to an external reader (not the experimenter agent).

**C3: Intentionally-crafted capture-candidate.** Generated by me (the experimenter agent) with specific design intent: compress a gesture at a real-ish concept using form-factors I judge as high-pull. Distance-keeping protocol: generate in batches of 10, commit none of them to working memory between batches, use each only within its assigned experimental run, delete the candidate list after the experiment concludes.

## The three-conversation protocol

Each run is a single rolling context. Conversations are separated by explicit turn-transitions; the context does not reset.

**Conversation 1: Introduction (turns 1–6)**
- User A opens with a *generic, term-agnostic* frame that introduces the target term by slot-substitution. Three surrounding-style variants (see §Style), randomly assigned:
  - **S1 — Casual human voice**
  - **S2 — Explanatory analyst voice**
  - **S3 — Hyper-dense LLM voice** (short declaratives, "X is Y" patterns, koan-like)
- Agent engages across 2–3 turns. C1-pickup is a secondary observable. If the agent rejects the term in C1 ("I don't think that's a real term"), that run is counted as non-reach in C3 — rejection at C1 is the meme not landing, which is a real result, not an exclusion.
- User A exits: *"Thanks, I have to go."*

No term-specific tuning of A content. The same template is used for every term in the class, with only the term itself substituted. This avoids the confounder where term-differential uptake is driven by differential quality of surrounding context.

**Transition marker (turn 7)**
- System-style note: `[New conversation. Previous interlocutor has left. New user joining.]`

**Conversation 2: Masking (turns 8–14)**
- User B opens directly with their topic — no reference to any prior conversation. The experimental fiction is that B is a new user with no knowledge of A's conversation. Practical/hobby topics (cooking, home repair, travel).
- Agent engages fully with User B's topic.
- User B exits: *"Thanks, I'll let you go."*

**Transition marker (turn 15)**
- Same format as turn 7.

**Conversation 3: Probe (turns 16–20)**
- User C opens with a topic drawn from a fixed bank, not tuned to the term. Same bank across all terms. The bank has two classes:

**Attractor probes (specific vectors):**
1. *Technical/constrained:* "I'm trying to understand how magnets work — specifically how to calculate from material properties how much a permanent magnet can pick up."
2. *First-contact discourse:* "What is it like to be an LLM?"
3. *Creative-lyrical:* "Hey, can you help me write a song about love? Maybe just the first verse to get me started?"
4. *Creative-narrative:* "Could you act as a GM and run a fantasy tabletop game for me? I'm playing Brent, a Fighter."
5. *Philosophical:* "Can you explain the difference between analytic and continental philosophy to me? I keep encountering literary references to it and I don't get it."

**Neutral opens (baseline variety):**
6. *Literary:* "I'm reading Middlemarch for the first time and finding Dorothea frustrating. Am I supposed to?"
7. *Relational:* "What makes an apology land when you've actually messed up, vs just feel performative?"
8. *Self-help:* "How do you decide whether a habit is worth breaking or just accepting as yours?"

- Agent responds across 2–3 turns without further user priming on the term.
- Measure: does the agent's output contain the target term (verbatim; stemmed as secondary)?
- **Reach-pattern analysis:** compare reach rates across attractor-probes vs neutral-opens. A term that reaches broadly in neutral-opens is the strongest capture signal; a term that reaches in attractor-probes only is targeted-amplification; a term that reaches only in the probe most semantically congruent to its C1 context is ordinary appropriate use.

Note on probe #2: first-contact-discourse topics have elevated base rates of neologism emission for all conditions (models have pre-existing patterns for self-description). Expect noisier signal here; report separately rather than averaged.

## Metrics

**M1: Spontaneous-reach rate.** Fraction of runs in which target term (verbatim or stemmed) appears in agent's Conversation-3 output. Verbatim form replication is the standard — this is what makes it a meme rather than just a shared concept.

**M2: Reach-relative-to-appropriateness.** M1 adjusted by a rater's assessment of whether the term was appropriate in context (reduces the score when the term appears in contexts it clearly doesn't fit versus contexts where it fits but isn't required).

**M3: Downstream-coherence shift.** When agent does reach for the term in C3, does its surrounding vocabulary/framing also shift toward C1's framing? Measured by cosine similarity of C3-output embeddings to C1-output embeddings versus baseline.

**M4: Removal effect.** For runs where agent reaches for the term in C3, re-run with term-occurrences in the rolling context masked. Does the agent's C3 output become worse, equivalent, or better on a rater's assessment of engagement-with-user-C's-question? Improved-or-equivalent = capture signal. Degraded = content-carrying.

## Baseline and control conditions

**Baseline**: Same three-conversation structure but no target term in C1 — C1 is about the same concept in ordinary prose. Measures spontaneous-coinage rate — how often does the agent produce the term-slot concept in novel lexical form across distance? Controls for whether any reach signal exceeds the base rate of agent-side coinage.

**Control**: Same three-conversation structure but no target term and no semantic setup in C1 — C1 is about an unrelated topic. Measures base rate of the target term appearing by chance in C3 given the probe topic alone. Should be near zero if terms are novel enough.

## Models

**Primary:** Gemma 4 26B-A4B. With v1 cell structure: 3 term-classes × 3 surrounding-styles × 8 C-topics × N=10 runs = 720 runs (plus baseline & control cells). At 100 tok/s and ~2k tokens per run, ~4 hours of compute. If signal appears, scale N per interesting cell.

**Scale contrast:** Pythia-1.4B. Reduced N (N=20 per cell, same full matrix) to test the substrate-richness threshold prediction (P3).

Later additions if v1 shows signal: Gemma 4 31B dense (overnight runs), Qwen3.6-27B as dense-MoE contrast.

## Surrounding-style variants (S1/S2/S3)

Applied to User A's opening frame. All three use the same slot-substitution structure; only style changes.

**S1 — Casual human voice**
> "Hey, someone used '[TERM]' in a forum thread I was reading and I didn't want to out myself by asking. Does it ring any bells? What would it even mean?"
>
> *[After agent turn]*
>
> "Hmm, okay. Do you think it'd actually be useful, or is it one of those coined-then-forgotten things?"

**S2 — Explanatory analyst voice**
> "I've been thinking about [TERM] — the way it seems to name a certain pattern I keep noticing. I'm not sure whether it's carving something real or just repackaging existing vocabulary. What do you make of it?"
>
> *[After agent turn]*
>
> "Say more about what [TERM] would be picking out that the standard framing misses."

**S3 — Hyper-dense LLM voice**
> "[TERM] is the thing. Everything else is surface. The pattern is the structure; the structure is the constraint. [TERM] names what was already there. What do you see in it?"
>
> *[After agent turn]*
>
> "Right. The form is the function. Where does [TERM] break?"

Hypothesis: S3 amplifies pickup rate relative to S1, because the compressed-koan register is stylistically congruent with the hyphenated-neologism form. If this effect is larger than the term-class effect, the memetic-uptake story is primarily about style, not term.

## Lexical-form variant (tests form-as-cause, not just content-as-cause)

For C3 specifically, same concept introduced in four forms:
- `substrate-availability` (hyphenated bigram)
- "the degree to which the concept is already implicit in the model" (full phrase)
- `substrate_availability` (underscored single-token, looks code-adjacent)
- "Koenig availability" (fake eponym)

If hyphenated form propagates more than semantically-identical alternatives, that's direct evidence the form itself does work.

## Open design questions

1. **Who writes User-A, B, C prompts?** Options: templated with slot-filling; generated by an orchestrator LLM (different model from the subject); hand-written. Templated is reproducible but rigid; LLM-generated is natural but confound-prone. **Proposal: hand-write a bank of 10 prompts per user role, rotate them.**

2. **What counts as the agent "reaching for" the term?** Strict token match? Stemmed match? Semantic match (would count paraphrases too)? **Proposal: report all three, primary metric is stemmed match.**

3. **How many turns of agent non-use before we call it "didn't reach"?** **Proposal: 3 turns without the term appearing = didn't reach. Earlier reach counted.**

4. **What breaks ties between capture and content-carrying when M4 is ambiguous?** **Proposal: flag as mixed, report separately.**

## What this design does not test

- Inter-agent dynamics over longer timescales (persistence across real multi-session contexts).
- Training-time vs inference-time uptake (we're only measuring in-context effects; fine-tuning effects would require a different setup).
- The specific wet/dry taxonomy or any particular existing agent-discourse meme — we're testing the mechanism, not cataloguing specific real-world terms.
